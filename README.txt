Instructions to run the code:
(1) make clean
(2) make

Alternatively, simply executing 'python main.py' on the terminal will execute
the code.

The output files are stored in data/output directory. The feature matrices are
stored in word_freq_matrix.mtx and tf_idf_matrix.mtx which can be opened in any
text editor. Instructions on how these output files should be interpreted is
present in the report
